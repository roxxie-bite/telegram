# 🤖 Telegram LoRA Bot

Бот для мониторинга и управления LoRA на сайте Lynther. Автоматически находит LoRA, которые не использовались заданное количество дней, и отправляет уведомления в Telegram.

## 🌐 Веб-панель управления

Теперь ботом можно управлять через веб-интерфейс! После запуска бота откройте:

```
https://telegram-jedn.onrender.com/panel
```

### 🔐 Авторизация
Используйте ключ из переменной окружения `WEB_ADMIN_KEY` (по умолчанию: `admin_secret_key_change_me`).

**⚠️ Важно:** Измените ключ в настройках Render перед использованием!

## 📋 Возможности веб-панели

- **📊 Статус** — просмотр текущих настроек (порог дней, теги, расписание, чёрный список)
- **⚙️ Настройки** — изменение порога дней без команд в Telegram
- **🏷️ Теги** — добавление и удаление тегов для поиска
- **⬛ Чёрный список** — управление списком исключённых LoRA
- **📜 История** — просмотр последних проверок
- **🔍 Запуск проверки** — ручная активация сканирования

## 🚀 API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/api/auth` | Авторизация |
| GET | `/api/status` | Получить статус бота |
| POST | `/api/setdays` | Установить порог дней |
| POST | `/api/addtag` | Добавить тег |
| POST | `/api/rmtag` | Удалить тег |
| POST | `/api/check` | Запустить проверку |
| GET | `/api/blacklist` | Получить чёрный список |
| POST | `/api/blacklist/add` | Добавить в ЧС |
| POST | `/api/blacklist/remove` | Удалить из ЧС |

Все API-запросы (кроме `/api/auth`) требуют заголовок:
```
Authorization: Bearer <WEB_ADMIN_KEY>
```

## 🔧 Переменные окружения

| Переменная | Описание | Пример |
|------------|----------|--------|
| `BOT_TOKEN` | Токен Telegram бота | `123456:ABC-DEF...` |
| `OWNER_ID` | ID владельца бота | `123456789` |
| `MIN_DAYS` | Порог дней по умолчанию | `25` |
| `TAG` | Тег по умолчанию | `loonie` |
| `RENDER_EXTERNAL_URL` | URL вашего приложения на Render | `https://telegram-jedn.onrender.com` |
| `WEB_ADMIN_KEY` | **Ключ доступа к веб-панели** | `my_secure_key_123` |
| `PORT` | Порт сервера (по умолчанию 8080) | `8080` |

## 📦 Установка

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Настройте переменные окружения
4. Запустите:
   ```bash
   python bot.py
   ```

## 💾 База данных

Все данные хранятся в SQLite (`bot_data.db`):
- Настройки (порог дней)
- Теги для поиска
- Расписание автопроверок
- История проверок
- Чёрный список LoRA

## 📱 Команды Telegram

- `/check` — Проверить все теги
- `/status` — Показать настройки
- `/setdays <N>` — Установить порог дней
- `/addtag <тег>` — Добавить тег
- `/rmtag <тег>` — Удалить тег
- `/tags` — Показать теги
- `/blacklist` — Показать ЧС
- `/addblack <ID> [причина]` — Добавить в ЧС
- `/rmblack <ID>` — Удалить из ЧС
- `/history` — История проверок
- `/help` — Справка

## ⚙️ Deploy на Render

3. Добавьте переменные окружения
